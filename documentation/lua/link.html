Lua API

Link

Link

Links are returned from Traversal requests starting with NodeGetRelationshipsIds

node_id

Retrieve the node id of the Link

Link.node_id

Example:


names = {}
ids = NodeGetRelationshipsIds("Node", "Max")
for k=1,#ids do
    v = ids[k]
    table.insert(names, NodePropertyGetById(v.node_id, "name"))
end
names

rel_id

Retrieve the relationship id of the Link

Link.rel_id

Example:


weights = {}
ids = NodeGetRelationshipsIds("Node", "Max")
for k=1,#ids do
    v = ids[k]
    table.insert(names, RelationshipPropertyGetById(v.rel_id, "weight"))
end
weights
